# 项目状态跟踪文档

## 1. 任务目标（固定）
开发一款接入Yahoo Finance的桌面端定投回测应用，支持对任意品种和投资组合进行带专业金融指标的定投策略回测，支持多种定投变种方法，提供明亮清新苹果风格的流畅UI体验，技术栈使用Tauri 2.0 + Vue 3 + Tailwind CSS + Lightweight Charts + SQLite。
