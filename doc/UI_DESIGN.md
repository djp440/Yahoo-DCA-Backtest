# 智能定投回测系统 - UI设计文档

> **版本**: v1.0  
> **设计风格**: 明亮清新 · 苹果设计规范  
> **技术实现**: Tailwind CSS + Vue 3 + 自定义动画系统  
> **适配要求**: 全响应式 · 多DPI支持 · 窗口任意缩放

---

## 1. 设计系统 (Design System)

### 1.1 色彩体系

主色调采用柔和的蓝色系，符合金融产品专业感的同时保持明亮清新的气质。

| 类别 | 色值 | 说明 | Tailwind 变量 |
|------|------|------|---------------|
| 主色 | `#007AFF` | 品牌主色，用于强调按钮、选中状态 | `blue-500` |
| 主色浅 | `#E1F0FF` | 主色背景、悬浮态 | `blue-100` |
| 主色深 | `#0051A8` | 按压态 | `blue-700` |
| 成功色 | `#34C759` | 正收益、成功状态 | `green-500` |
| 警告色 | `#FF9500` | 警告、中等风险 | `orange-500` |
| 危险色 | `#FF3B30` | 负收益、错误、高风险 | `red-500` |
| 上涨色 | `#FF3B30` | K线上涨、正收益 | `red-500` (符合A股习惯) |
| 下跌色 | `#34C759` | K线下跌、负收益 | `green-500` |

**中性色**:
| 色值 | 说明 | 用途 |
|------|------|------|
| `#FFFFFF` | 纯白 | 页面背景、卡片背景 |
| `#F8F9FA` | 浅灰 | 次要背景、输入框背景 |
| `#E9ECEF` | 边框灰 | 边框、分割线 |
| `#ADB5BD` | 次要文字 | 说明文字、辅助信息 |
| `#495057` | 主要文字 | 正文、标题 |
| `#212529` | 深色文字 | 强调文本、标题 |

### 1.2 字体系统

采用苹果系统字体优先，保证原生体验。

```css
font-family: -apple-system, BlinkMacSystemFont, "SF Pro", "Inter", sans-serif;
```

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|

### 1.3 间距系统

基于8px基准网格，与Tailwind默认体系完全对齐。

## 2. 整体布局规范

### 2.1 布局结构

采用你选择的 **顶部菜单栏 + 下方工作区** 布局，结构如下：

```
┌───────────────────────────────────────────────────────────┐
│  顶部导航栏 (高度: 56px)                                   │
│ ┌─────────┬───────────────────────┬─────────────────────┐ │
│ │ Logo    │  主菜单 (回测/数据/历史)  |  全局操作(设置/帮助)   │ │
│ └─────────┴───────────────────────┴─────────────────────┘ │
├───────────────────────────────────────────────────────────┤
│  Tab 标签栏 (高度: 44px)  [可选，多任务时显示]              │
│  ┌──────┬──────┬──────┬────────────────────────────────┐  │
│  │ 回测1│ 回测2│ 回测3│  + 新建                        │  │
│  └──────┴──────┴──────┴────────────────────────────────┘  │
├───────────────────────────────────────────────────────────┤
│                                                           │
│  工作区 (剩余高度，自适应)                                 │
│  ┌─────────────────────────────────────────────────────┐  │
│  │                                                     │  │
│  │                页面内容区域                         │  │
│  │                                                     │  │
│  └─────────────────────────────────────────────────────┘  │
│                                                           │
└───────────────────────────────────────────────────────────┘
```

### 2.2 布局分区说明

| 区域 | 功能 |
|------|------|
| **顶部导航栏** | 全局导航、菜单、设置、搜索 |
| **Tab标签栏** | 多任务管理，每个回测/页面一个标签，支持关闭、拖拽排序 |
| **工作区** | 主要内容展示区域，根据不同页面动态变化 |

### 2.3 响应式规则

- **最小窗口宽度**: 1024px
- **支持任意比例缩放**: 所有组件自适应窗口大小
- **高DPI适配**: 支持2x/3x Retina屏幕，所有图标使用SVG格式
- **窗口缩放**: 所有元素支持系统缩放比例，文字大小自动适配

## 5. 交互与动画规范

### 5.1 动画系统

遵循你要求的 **丰富动感 (弹性动画、视差效果)**，提供流畅的体验。

**动画基础配置**:
```javascript
const animations = {
  // 缓动函数
  easing: {
    standard: 'cubic-bezier(0.4, 0, 0.2, 1)',    // 标准缓动
    decelerate: 'cubic-bezier(0, 0, 0.2, 1)',    // 减速
    accelerate: 'cubic-bezier(0.4, 0, 1, 1)',    // 加速
    spring: 'cubic-bezier(0.175, 0.885, 0.32, 1.275)', // 弹性效果
  },
  // 时长
  duration: {
    fast: '150ms',    // 小元素、点击反馈
    normal: '250ms',  // 页面切换、组件显示隐藏
    slow: '350ms',    // 大动画、模态框弹出
  }
}
```

### 5.2 动画场景

| 场景 | 效果 |
|------|------|
| **页面切换** | 淡入淡出 + 轻微Y轴位移 (从下往上滑入) |
| **组件加载** | 骨架屏渐入 → 内容渐入 + 轻微弹性缩放 |
| **按钮交互** | 按下时轻微缩小(0.98倍)，松开时弹性恢复 |
| **卡片悬浮** | 上移2px + 阴影加深，过渡动画200ms |
| **模态框弹出** | 从屏幕中心缩放弹出(从0.9到1) + 背景渐暗 |
| **下拉菜单** | 展开时从上方滑入 + 淡入 |
| **图表加载** | 数据从左到右渐进式绘制动画 |
| **数字变化** | 滚动动画，从旧数字平滑过渡到新数字 |
| **进度条** | 平滑填充动画，带轻微弹性效果 |

### 5.3 微交互细节

- 所有可点击元素 hover 时有明确的视觉反馈
- 操作后即时给出成功/失败提示，无延迟感
- 加载状态显示骨架屏或进度指示器，避免空白
- 表单验证错误时元素轻微抖动动画提示
- 滚动页面时导航栏有轻微视差效果

## 6. 组件设计规范

### 6.1 按钮组件

| 类型 | 样式 |
|------|------|
| **主按钮** | 背景主色 #007AFF，文字白色，圆角8px，hover时背景加深10%，按下时加深20% |
| **次按钮** | 背景白色，边框主色，文字主色，hover时背景主色浅 #E1F0FF |
| **文字按钮** | 无背景无边框，文字主色，hover时背景浅灰 |
| **危险按钮** | 背景红色 #FF3B30，文字白色 |

### 6.2 表单组件

- 输入框高度：36px，圆角8px，边框#E9ECEF，focus时边框主色 + 2px主色阴影
- 下拉框：高度36px，展开时下拉菜单带阴影，选项hover背景浅灰
- 复选框/单选框：苹果风格，选中时主色填充，未选中时灰色边框
- 日期选择器：明亮风格，选中日期主色背景

### 6.3 卡片组件

- 圆角12px，背景白色，shadow-sm，hover时shadow-md + 上移1px
- 内边距：16px (md间距)
- 卡片标题与内容间距12px

### 6.4 模态框/弹窗

- 圆角16px，背景白色，shadow-xl
- 宽度最大800px，最小400px，自适应内容
- 弹出动画：弹性缩放 + 淡入
- 背景遮罩：半透明黑色 rgba(0,0,0,0.4)

### 6.5 表格组件

- 行高40px，边框浅灰 #E9ECEF
- hover行背景浅灰 #F8F9FA
- 选中行背景主色浅 #E1F0FF
- 表头背景 #F8F9FA，字重600
- 支持固定表头、固定首列、虚拟滚动

## 8. 实现建议

### 8.1 Tailwind 配置

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#007AFF',
        success: '#34C759',
        warning: '#FF9500',
        danger: '#FF3B30',
        up: '#FF3B30',
        down: '#34C759',
      },
      fontFamily: {
        sans: ['-apple-system', 'BlinkMacSystemFont', 'Inter', 'sans-serif'],
      },
      borderRadius: {
        'card': '12px',
        'modal': '16px',
      },
      animation: {
        'spring': 'spring 0.3s ease-out',
        'fade-in': 'fadeIn 0.25s ease-out',
        'slide-up': 'slideUp 0.3s ease-out',
      },
      keyframes: {
        spring: {
          '0%': { transform: 'scale(0.95)' },
          '100%': { transform: 'scale(1)' },
        },
        fadeIn: {
          '0%': { opacity: '0' },
          '100%': { opacity: '1' },
        },
        slideUp: {
          '0%': { transform: 'translateY(10px)', opacity: '0' },
          '100%': { transform: 'translateY(0)', opacity: '1' },
        },
      },
    },
  },
}
```

### 8.2 动画库选择

推荐使用内置的Vue Transition + 自定义CSS动画，复杂动画可以引入：
- `@vueuse/motion` - 轻量级Vue动画库，支持视差效果
- `framer-motion` (可选) - 功能丰富的动画库，适合复杂交互
